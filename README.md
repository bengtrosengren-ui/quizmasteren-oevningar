# 🐍 Python-övningar — Quizmästaren

Välkommen! Det här repot innehåller 15 övningar som tränar de Python-koncept
du behöver för att koda klart Quizmästaren-projektet.

---

## 📂 Struktur

```
ovningar/       ← Här skriver DU din kod
tester/         ← Automatiska tester (redigera inte dessa)
```

### Övningarna är uppdelade i 5 ämnen:

| Nr    | Ämne                        | Filer                      |
|-------|-----------------------------|----------------------------|
| 01–03 | Listor & enumerate          | ovning_01–03.py            |
| 04–06 | Dictionaries                | ovning_04–06.py            |
| 07–09 | Returnera & använda värden  | ovning_07–09.py            |
| 10–12 | while-loopar                | ovning_10–12.py            |
| 13–15 | Strängmetoder               | ovning_13–15.py            |

---

## 🚀 Kom igång

### 1. Klona repot

```bash
git clone <din-classroom-länk>
cd quizmasteren-ovningar
```

### 2. Installera pytest

```bash
pip install pytest
```

### 3. Öppna i VS Code

```bash
code .
```

---

## ✅ Köra tester

### Kör ALLA tester på en gång:
```bash
pytest
```

### Kör tester för en specifik övning:
```bash
pytest tester/test_ovning_01.py
```

### Kör med mer detaljerad output:
```bash
pytest -v
```

---

## 📝 Så här jobbar du

1. Öppna `ovningar/ovning_01.py`
2. Läs uppgiftsbeskrivningen i kommentarerna
3. Skriv din lösning där det står `# DIN KOD HÄR`
4. Kör `pytest tester/test_ovning_01.py` för att testa
5. Får du grönt ✅? Gå vidare till nästa!

> 💡 **Tips:** Kör `pytest -v` för att se exakt vilka tester som passerar.
